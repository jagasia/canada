**Proposed Training Plan**

**Week 1 (8–10 Jul) – Backend Foundations**  
**Day 1 – Java & Spring Boot Basics**  
- Java refresher: OOP, collections, streams  
- Spring Boot setup: project structure, annotations  
- REST API basics: controllers, request/response mapping  
- Lab: Build "Hello API"  
 
**Day 2 – REST API Design & Data Access (Part 1)**  
- REST API design: CRUD, versioning, error handling  
- JPA/Hibernate basics: entities, relationships  
- Repository pattern & query methods  
- Lab: CRUD endpoints with DB integration  
 
**Day 3 – Data Access (Part 2)**  
- Transaction management  
- Performance tuning (lazy vs eager loading, caching)  
- Lab: SQL DB integration project

**Week 2 (13–17 Jul) – Security + Frontend**  c
**Day 4 – Spring Security Basics**

**Day 5 – Vue.js Fundamentals (Part 1)**  
- Project setup: Vue CLI / Vite  
- Vue instance & single‑file components  
- Template syntax: directives (`v-bind`, `v-if`, `v-for`, `v-model`)

**Day 6 – Vue.js Fundamentals (Part 2)**  
- Reactivity system: `ref`, `reactive`, `computed`, `watch`

**Day 7 – Vue.js Advanced Concepts & Testing**  
- State management: Vuex / Pinia

**Day 10 – Databases (NoSQL)**

- Lab: Hybrid SQL + MongoDB app  
 
**Day 11 – Automated Testing & Quality Gates (Part 1)**
