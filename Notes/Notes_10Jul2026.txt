in controller,
	every web method returns view (or model and view)
	ex:
		return "index";				//index is a view


in rest controller,
	every web method returns the model alone.
	ex:
		return employee;			//employee is an object of Employee class. 





even in a controller method,
	@ResponseBody annotation to convert the behavior of a method to return the model. ie., content.




@RestController	=	@Controller + @ResponseBody
