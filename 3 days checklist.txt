Definition

A transaction ensures a group of database operations either all succeed or all fail.

Example

2. Lazy vs Eager Loading
Definition

Lazy loading fetches related data only when needed, while eager loading fetches it immediately.

Example
@OneToMany(fetch = FetchType.LAZY)

@ManyToOne(fetch = FetchType.EAGER)
Checklist
□ LAZY
□ EAGER
□ Default fetch types
□ Performance implications
Hands-on
