simple-employee-app
===================

A minimal Java 21 Maven project that demonstrates a simple Employee class and an App main that prints "Hello World" and creates an Employee instance.

Quick run:

```powershell
cd "D:\Jag\Empower\CanadaLife\Java\simple-employee-app"
mvn package
java -jar target\simple-employee-app-0.0.1-SNAPSHOT.jar
```
