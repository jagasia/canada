# Collection Operations Task

This project provides a hands-on task to demonstrate understanding of Java Collections (`List`, `Set`, and `Map`). The `CollectionsOperations.java` file contains a skeleton where you need to implement various operations as described in the `// TODO:` comments.

## Project Structure

- `pom.xml`: Maven build configuration.
- `src/main/java/com/example/collections/CollectionsOperations.java`: The main class where the collection operations are to be implemented.

## How to Run

1.  **Prerequisites**: Ensure you have Java Development Kit (JDK) 17 or newer and Apache Maven installed.

2.  **Clone or Download**: Get this project to your local machine.

3.  **Navigate to Project Directory**: Open your terminal or command prompt and change your current directory to the root of this project (where `pom.xml` is located).

4.  **Implement the Task**: Open `src/main/java/com/example/collections/CollectionsOperations.java` and fill in the code according to the `// TODO:` comments.

5.  **Compile and Run**: Use Maven to compile and run the application:

    ```bash
    mvn clean install
    mvn exec:java
    ```

    This will execute the `main` method in `CollectionsOperations.java` and print the results to the console.
