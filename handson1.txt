# Hands-on Exercise: Employee Management System

## Objective

Develop a simple Employee Management System using Object-Oriented Programming concepts in Java.

---

## Requirements

### Step 1: Create an abstract class `Employee`

It should contain the following private instance variables:

* id
* name
* department
* salary

Provide:

* Default constructor
* Parameterized constructor
* Getters and Setters
* `toString()` method

Declare the following abstract method:

```java
public abstract double calculateBonus();
```

---

### Step 2: Create the following subclasses

#### Developer

Bonus = 20% of salary

---

#### Tester

Bonus = 15% of salary

---

#### Manager

Bonus = 30% of salary

Each class should override the `calculateBonus()` method.

---

### Step 3: Create objects

Create an array of **10 Employee references**.

Store a mixture of:

* Developers
* Testers
* Managers

using runtime polymorphism.

Example:

```java
employees[0] = new Developer(...);
employees[1] = new Tester(...);
employees[2] = new Manager(...);
```

---

### Step 4: Display Employee Details

Using a loop, display

* ID
* Name
* Department
* Salary
* Bonus

---

### Step 5: Store in a Collection

Create a `HashSet<Employee>`.

Add all employees from the array into the set.

Display the number of unique employees.

---

### Step 6: Search

Ask the user to enter an Employee ID.

Search the employee in the array.

If found, display complete details along with the calculated bonus.

Otherwise display

```
Employee Not Found
```

---

## Boilerplate Code

### Employee.java

```java
public abstract class Employee {

    private int id;
    private String name;
    private String department;
    private double salary;

    public Employee() {
    }

    public Employee(int id, String name, String department, double salary) {
        this.id = id;
        this.name = name;
        this.department = department;
        this.salary = salary;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public String getDepartment() {
        return department;
    }

    public void setDepartment(String department) {
        this.department = department;
    }

    public double getSalary() {
        return salary;
    }

    public void setSalary(double salary) {
        this.salary = salary;
    }

    @Override
    public String toString() {
        return "ID : " + id +
               ", Name : " + name +
               ", Department : " + department +
               ", Salary : " + salary;
    }

    public abstract double calculateBonus();
}
```
